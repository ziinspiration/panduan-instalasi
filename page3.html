Panduan Instalasi WebGIS

3. Import database

  • Pertama silahkan buka browser anda (misalnya: chrome, microsoft edge)

  • Arahkan url dengan mengarahkan ke localhost/phpmyadmin/, atau anda klik disini

  • Selanjutnya masukan username & password (secara default, username adalah root dan password tidak ada) , dan klik Log in

  • Jika sudah masuk ke halaman PHPmyadmin, silahkan klik new yang ada di pojok kiri atas

  • Selanjutkan buatkan nama database yaitu webgis_cilegon dan klik create

  • Selanjutnya klik database dengan nama webgis_cilegon

  • Jika sudah masuk ke database dengan nama webgis_cilegon, lalu klik import

  • Pada tampilan import silahkan klik Choose file dan pilih file pada folder website yang di download pada awal tadi, pilih file dengan format sql
    jika anda belum mendownload silahkan klik Disini

  • Jika sudah, silahkan scroll ke bawah dan klik import
