    Download Here
    • 1. Download WEB Server

  • Silahkan kunjungi laragon.org, laragon adalah sebuah web server seperti XAMPP yang di dalamnya sudah meliput Apache, Nginx, MySQL

  • Lalu silahkan klik download dengan mengklik Download Laragon - Full (173 MB)

  • Setelah terdowload cari hasil download, dan klik untuk menginstal

  • Setelah terinstal, silahkan klik OK

  • Arahkan file ke Disk C

  • Setelah nya silahkan klik next hingga selesai

  • Jika sudah, klik Yes, restart computer now

  • Selamat laragon telah terinstal :)
