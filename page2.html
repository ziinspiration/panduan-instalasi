Panduan Instalasi WebGIS

2. Konfigurasi WEB Server

  • Silahkan buka aplikasi laragon yang sudah terinstal tadi

  • Selanjutnya klik MENU yang ada di pojok kiri atas

  • Selanjutnya klik Tolls lalu pilih Quick Add, dan pilih *phpmyadmin

  • Setelah PHPMyadmin sudah terinstal, silahkan hidupkan Web Server dengan mengklik tombol Mulai semua / Start All
