3. Instalasi website

  • Pertama buka folder website yang sudah di download, dan pindahkan folder website ke folder laragon, biasanya folder laragon ada di Disk C

  • Selanjutkan masukkan folder ke dalam folder laragon > www , dan copy kan didalam folder www

  • Instalasi website sudah selesai, anda bisa mengakses nya dengan membuka pada chrome dengan url localhost/website/ atau klik disini
